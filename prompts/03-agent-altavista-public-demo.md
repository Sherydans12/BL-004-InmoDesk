# Prompt para agente — Sitio público demo Altavista Propiedades

Actúa como agente de desarrollo/diseño para el repo independiente **Altavista Propiedades**.

Este repo NO es el producto principal BL-004. Es una demo pública a medida para mostrar cómo podría verse una web inmobiliaria conectada al panel BL-004 InmoDesk.

Objetivo:

Crear una web pública profesional para una corredora ficticia llamada Altavista Propiedades.

Pantallas/secciones:

1. Home.
2. Listado de propiedades.
3. Detalle de propiedad.
4. Publica con nosotros.
5. Contacto.

Debe mostrar:

- Buscador de propiedades.
- Filtros.
- Propiedades destacadas.
- Fichas visuales.
- Formulario de interés por propiedad.
- Botón WhatsApp.
- Sección para captar propietarios.

Relación con BL-004:

- Consumir o simular propiedades publicadas.
- Enviar o simular leads hacia el admin.
- No acoplar el diseño público al admin.

No incluir:

- Panel administrativo.
- Login de admin.
- Funciones internas de BL-004.
