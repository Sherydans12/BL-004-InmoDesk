# 05 — Módulos principales

## Dashboard

Vista inicial del panel.

Debe mostrar resumen operativo y comercial:

- Propiedades activas.
- Propiedades publicadas.
- Leads nuevos.
- Visitas agendadas.
- Leads pendientes de contacto.
- Propiedades sin movimiento.
- Comisión potencial estimada.

Accesos rápidos:

- Nueva propiedad.
- Nuevo lead.
- Agendar visita.
- Ver sitio público.
- Configurar publicación.

## Propiedades

Módulo central del sistema.

Debe permitir:

- Crear propiedades.
- Editar propiedades.
- Publicar/ocultar propiedades.
- Marcar como destacadas.
- Gestionar fotos.
- Ver leads asociados.
- Ver visitas asociadas.
- Mantener datos internos separados de datos públicos.

## Galería

Debe permitir:

- Foto principal.
- Fotos secundarias.
- Orden de fotos.
- Texto alternativo.
- Video o tour virtual en fase posterior.

## Leads

Debe capturar interesados desde:

- Sitio público.
- Registro manual.
- WhatsApp manual.
- Referido.
- Portal externo manual.

Debe permitir:

- Asignar agente.
- Cambiar estado.
- Agendar visita.
- Registrar notas.
- Ver propiedad consultada.

## Pipeline

Vista Kanban para seguimiento comercial.

Estados:

1. Nuevo.
2. Contactado.
3. Visita agendada.
4. Visitó.
5. En negociación.
6. Cerrado.
7. Perdido.

## Agenda

Gestión básica de visitas.

Debe permitir:

- Crear visita.
- Reagendar.
- Cancelar.
- Marcar realizada.
- Relacionar visita con lead y propiedad.

## Propietarios

Gestión de dueños de propiedades.

Debe permitir:

- Registrar contacto.
- Asociar propiedades.
- Registrar condiciones comerciales.
- Registrar comisión acordada.
- Mantener notas internas.

## Configuración pública

Conecta el panel con la web pública.

Debe permitir:

- Configurar marca.
- Configurar datos de contacto.
- Configurar redes sociales.
- Obtener código/API de integración.
- Definir campos visibles.
