# Backlog v0 — Admin Panel

## Épicas

### EPIC-01 Dashboard

- Como usuario quiero ver métricas principales para entender el estado de mi operación inmobiliaria.
- Como usuario quiero ver leads recientes para actuar rápido.
- Como usuario quiero acceder rápido a crear propiedad, lead o visita.

### EPIC-02 Propiedades

- Como usuario quiero crear propiedades con datos comerciales e internos.
- Como usuario quiero publicar u ocultar propiedades.
- Como usuario quiero marcar propiedades como destacadas.
- Como usuario quiero filtrar propiedades por estado, tipo, operación y comuna.

### EPIC-03 Leads

- Como usuario quiero ver interesados captados desde la web.
- Como usuario quiero cambiar el estado de un lead.
- Como usuario quiero asignar un lead a un agente.
- Como usuario quiero agendar una visita desde un lead.

### EPIC-04 Pipeline

- Como usuario quiero ver leads en un pipeline comercial.
- Como usuario quiero mover leads entre estados.

### EPIC-05 Agenda

- Como usuario quiero registrar visitas.
- Como usuario quiero cambiar el estado de una visita.

### EPIC-06 Propietarios

- Como usuario quiero registrar propietarios.
- Como usuario quiero asociar propiedades a propietarios.

### EPIC-07 Configuración pública

- Como usuario quiero configurar datos públicos de mi corredora.
- Como usuario quiero ver un código de integración conceptual.
