# Prompt para agente de diseño UI — BL-004 InmoDesk

Actúa como diseñador UI/UX para BL-004 InmoDesk.

Debes diseñar una experiencia de panel inmobiliario premium, clara y comercial, evitando apariencia de ERP pesado.

Objetivo visual:

Una oficina inmobiliaria digital moderna para corredores e inmobiliarias pequeñas/medianas.

Pantallas a diseñar primero:

1. Login.
2. Dashboard.
3. Propiedades.
4. Crear/editar propiedad.
5. Detalle de propiedad.
6. Leads.
7. Pipeline.
8. Agenda.
9. Propietarios.
10. Configuración pública.

Principios:

- Fondo claro.
- Sidebar lateral.
- Cards limpias.
- Estados con chips.
- Fotos inmobiliarias protagonistas.
- Formularios divididos por secciones.
- Lenguaje comercial inmobiliario.
- Acciones rápidas visibles.

No diseñes el sitio Altavista dentro de este mismo paquete salvo que se pida explícitamente, porque será repo/carpeta aparte.
