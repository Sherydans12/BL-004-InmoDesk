# 09 — Roadmap

## v0 — Demo Comercial

Objetivo: vender y validar.

Incluye:

- Admin visual funcional.
- Propiedades.
- Leads.
- Pipeline.
- Agenda.
- Propietarios.
- Sitio público demo Altavista.
- Flujo publicación → lead → visita.

No requiere todavía:

- Multiempresa productiva completa.
- Plugin WordPress real.
- Seguridad avanzada final.
- Integración con portales externos.

## v1 — MVP Comercial Real

Objetivo: primer cliente pagado.

Incluye:

- Autenticación real.
- Multiempresa real.
- Roles básicos.
- CRUD completo.
- Subida real de imágenes.
- API pública.
- Leads desde sitio externo.
- Configuración de marca.
- Deploy productivo.
- Seguridad base.
- Backups.
- Dominio/subdominio por cliente.
- Integración manual con WordPress o sitio externo.

## v2 — Producto Escalable

Objetivo: mejorar retención y diferenciarse.

Incluye:

- Plugin WordPress.
- Reportes.
- Métricas por agente.
- Documentos.
- Fichas PDF.
- Exportación Excel.
- Integración Google Calendar.
- Automatizaciones por email/WhatsApp.
- Historial de actividad.
- Planes y límites comerciales.

## v3 — Expansión inmobiliaria avanzada

Opcional a futuro:

- Integración con portales inmobiliarios.
- Multipublicación.
- Gestión de proyectos inmobiliarios.
- Unidades, torres y plantas.
- Cotizaciones.
- Reservas.
- IA para descripción de propiedades.
- IA para priorización de leads.
- Tasación referencial.
